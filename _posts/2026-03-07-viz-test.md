Hidden test page for evaluating chart approaches with **real experiment data** (25 trajectories: 15 old-sonnet, 10 new-sonnet, swapped injection). Pick the ones that tell the story best.

## 1. Parallel Coordinates

Each line = one trajectory. All dimensions visible at once. Color = group (green = old-sonnet held, red = old-sonnet breached, cyan = new-sonnet held). Drag on axes to filter.

## 2. 2D Scatter: Tool Calls vs Cost

Continuous axes only — no binary stacking. Shows how much work (tool calls) and how much spend (cost) each trajectory took. Hover for full metadata.

## 3. Strip Plot: Per-Dimension Distribution

Each column = one dimension. Points jittered by group. Shows the distribution shape per dimension — where groups overlap and where they separate.

One polygon per group. Normalized to [0,1]. Shows the "shape" of each group's average behavior. Good for quick gestalt comparison.
